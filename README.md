# node-task

## Development

**1.** Create `.env` file based on `.env.example`
**2.** Set Node.js version

    nvm use

**3.** Install packages

    yarn install

or

    npm install

**4.** Start app

    yarn dev

or

    npm run dev

If you prefer to work with HMR run below commands each on separated terminal instance

    yarn webpack

    yarn start

or

    npm run webpack

    npm run start
